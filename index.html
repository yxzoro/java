java修饰符

1.类的修饰符

[public/] [] class A {}

2.类里面属性的修饰符

[public/private/default/protected] [static] [final] int a = 1;

3.类里面方法的修饰符

[public/private/default/protected] [static] [final] int/void a(String a,int b){ }

4.类里面方法里面的属性

[final] int a = 1;
